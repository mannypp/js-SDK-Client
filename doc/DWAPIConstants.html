Class: DWAPIConstants

DWAPIConstants

new DWAPIConstants()

The DWAPIConstants class. This class contains a series of constants used throughout the API. Call DWAPIConstants.get() to retrieve the value of the constant. For example: DWAPIConstants.get('COUNT') will cause the string 'count' to be returned.
