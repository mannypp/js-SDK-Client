DWAPIManager

new DWAPIManager()

The DWAPIManager class. This class manages connections to the server. In order to initialize the manager, make the following call: DWAPIManager.getInstance(server-name, site-name, client-id); The above code will initialize the manager using server, site, and client id specified. Note that you do not need to hold onto a reference to the manager class created. To retrieve the manager in your code later, just call: DWAPIManager.getInstance();
